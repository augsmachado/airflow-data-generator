## 🔧 Installation

### 1. Clone Repository

```bash
git clone https://github.com/augsmachado/airflow-data-generator.git
cd airflow-data-generator
```

### 🚦 Running the Application

```bash
docker compose up -d
```

Verify if containers are running

```bash
docker ps
```

### 🔍 Testing

```bash
pytest
```
